# php实现注册登陆留言板
# 1.运行环境
> php7.0+mysql5.5+apache2.7 

# 2.代码结构
- add.php
> 注册后insert页面
- ckly1.php
> 查看留言的页面
- insertdb.php
> 留言成功后跳转
- insertliuyan.php
> 登陆后跳转的留言板
- logout.php
> 注销功能
- success.php
> 验证登陆功能
